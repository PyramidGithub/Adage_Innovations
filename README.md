## AI-DASHBOARD: THIS IS A PROGRAM MGMT. INTERACTIVE DASHBOARD 
- DATA ARCHITECTURE INCLUDES A LARGE SEMANTIC MODEL ACCESSED VIA A DATA LAKE UTILIZING CACHING STRATAGIES, DYNAMIC URL'S, & OPTIMISED GEO-JSON PREPROCESSED IN OUR HgO DATAWAREHOUSE
- PRESENTING THE USA CONTINIUM OF CARE-(CoC) HOMELESSNESS & SHELTER PROGRAM - 50 STATES + TERRTORIES
- 15 YEARS OF HISTORICAL DATA COMBINED WITH RACE, AGE, & LITTANY OF OTHER SOCIO-DEMOGRAPHIC ATTRIBUTES AS WELL AS CENSUS DATA
- PLEASE NOTE FOR SAFETY REASONS CERTAIN DATUM RELATED TO FACILITY NAMES, IDENIFIERS, ADDRESS, OR LOCATION ARE OMITTED, COULD BE OBFISCATED OR EVEN SPOOFED AFFECTING SOME SUBTOTALS

#### FEATURES

- A CUSTOM GRAPH-DB VIEW - EDGE/NODE WITH HOVER POPUPS
- MULTI LAYERED GIS MAP - 22 ZOOM LEVESS + PITCH & TILT + MAPTOOLS - 3 WAYS TO MANUALL SELECT DATA + MANY DIFFERENT OPTIONS FOR THE BASE LAYER INCLUDING ACCESSABILITY OPTIONS
- POINT DATA CLUSTERS DEPENDENT ON ZOOM/FILTER SELECTIONS, THE CHOROPHYLMAP LAYER USES A CUSTOM GEOMETRY  REFERENCE LAYER FEATURE
- ASSORTED KPI VISUALS W/ ADDITIONAL HOVER & DRLL DOWN CAPABILITIES

- INEGRATED WITH MS FABRIC & AZURE DATA LAKE, DATA GOVERNANCE, DATA SECURITY/SENSITIVITY, EVENT PROCESSING & MONITORING W/ REPORTING
- INITIALIZES & LOADS IN ~ 1 SEC.
- PORTABILITY FEATURES LIKE EMBEDCODES, LINKS, & A QR CODE FOR MOBILE DEVICES
